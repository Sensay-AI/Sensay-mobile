## How to run
1. create your .env file in the repo
```dotenv
AUTH0_DOMAIN=<-your-auth0-domain->
AUTH0_CLIENT_ID=<-your-auth0-client-id->
```
2. Start react native metro server
```shell
npm run start
```
3. Start ios app
```shell
npm run ios
```
4. Start android app
```shell
npm run ios
```
